# Server 

## Основной стек
- **NestJS**
- **TypeScript**
- **Express**
- **Swagger**

## Запуск локально 

### Установка зависимостей
```bash
npm install
```

### Запуск
```bash
npm run start:dev
```
Сервер будет доступен по адресу: `http://localhost:3000`

### Сборка проекта
```bash
npm run build
```
