# Server 

## Основной стек
- **NestJS**
- **TypeScript**
- **Express**
- **Swagger**

### Установка зависимостей
```bash
npm install
```

### Запуск
```bash
npm run start:dev
```
Сервер будет доступен по адресу: `http://localhost:3000`
Swagger UI: `http://localhost:3000/api-docs`

### Сборка проекта
```bash
npm run build
```
